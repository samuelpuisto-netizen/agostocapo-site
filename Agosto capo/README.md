# Agosto & Capo

Simple multi-page artist website built with plain HTML and CSS.

## Files

- `index.html` - etusivu
- `tietoa.html` - artistien tiedot
- `galleria.html` - kuvat
- `live.html` - videot ja Spotify
- `styles.css` - shared site styling
- `script.js` - empty placeholder
- `assets/` - local photos and video used by the site

## Run

Open `index.html` in your browser.

If you want a local server, from this folder run:

```powershell
python -m http.server 8000
```

Then visit `http://localhost:8000`.

You can also double-click `start.bat` to open the site and start the local server automatically using the shared `.venv` Python.

For the mobile mockup, double-click `open-mobile-preview.bat` to open `mobile-preview.html` through a local server.

If your editor preview gives a DNS error, double-click `open-site.bat` to open `index.html` directly in your normal browser.
